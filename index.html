What are Lists?

A list is a sequence type data structure

Think of a list as a collection of values


['eggs', 'milk', 'apples', 2, 5.7, {}, [10, 20]]

['eggs', 'milk', 'apples', 2, 5.7, {}, [10, 20, [1, 2, 3]]]

['eggs', 'milk', 'apples', 2, 5.7, {}, [10, 20, [1, 2, [4, 5, 6]]]]

It is lists all the way down

How do you create a list?

empty_list = []

my_list = list()

shopping_list = ['eggs', 'milk', 'honey']

numbers_list = [x for x in range(100) if x % 2 == 0]
How do you create a list?

shopping_list = list()

shopping_list.append('eggs')
shopping_list.append('milk')
shopping_list.append('honey')
shopping_list.append('cookies')

print(shopping_list)
>>> ['eggs', 'milk', 'honey', 'cookies']
List comprehension

shopping_list = ['eggs', 'milk', 'honey', 'cookies']
quantities = [12, 1, 1, 36]

sorted_shopping_list = sorted([
    (quantity, item)
    for item, quantity
    in zip(shopping_list, quantities)
])

print(sorted_shopping_list)
>>> [(1, 'honey'), (1, 'milk'), (12, 'eggs'), (36, 'cookies')]
List items

We use indices to look up certain items inside the list


arr = ['eggs', 'milk', 'apples', 2, 5.7, [10, 20]]

print(arr[2])

>>> apples
Lists are mutable - aka "changeable"

As opposed to strings, strings are not mutable


arr = ['eggs', 'milk', 'apples', 2, 5.7, [10, 20]]

arr[2] = 'honey'

print(arr)
Membership operator "in"

shopping_list = ['eggs', 'milk', 'apples']

'milk' in shopping_list
>>> True

'honey' in shopping_list
>>> False

'cookies' not in shopping_list
>>> True
List concatenation

a = [1, 2, 3]
b = [4, 5, 6]

c = a + b

print(c)
>>> [1, 2, 3, 4, 5, 6]
List slicing

Very similar to how slicing works with strings

Defined as s[start:stop:step]

Reminder: up to but not including..


arr = [1, 2, 3, 4, 5, 6]

arr[:3]
>>> [1, 2, 3]

arr[3:]
>>> [4, 5, 6]

arr[::-1]
>>> [6, 5, 4, 3, 2, 1]
Copying lists with [:]

l1 = [1, 25, 65, 34, 17, 9]

l2 = l1

id(l1), id(l2)

l3 = l1[:]

id(l3)

l1 == l2 == l3 # True

l1 is l2 # True
l1 is l3 # False

The range function

Defined as range(start, stop, step) or simply range(X)


shopping_list = ['eggs', 'milk', 'apples']

num_items = len(shopping_list)

for index in range(num_items):
    print(index+1, shopping_list[index])
