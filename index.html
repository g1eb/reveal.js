url = 'https://data-collection.com/api?page_num=1'

    if not url:
        print('Please enter a valid url')
    else:
        response = requests.get(url)
        data = response.json()
        return data

Types of loops

In summary, there are three types of loops:

  1. Indeterminate
  2. Determinate
  3. Infinite
Loop Patterns

Typically, there are three parts to a loops:

  1. Initialization
  2. Computation
  3. Comparison
Loop Terminology
  • initialization - an assignment that gives an initial value to a variable that will be updated
  • iteration - a single cycle executing statements inside the loop
  • counter - a variable used to count the number of iterations in a loop
  • increment - an update that increases the value of a variable (often by one)
  • decrement - an update that decreases the value of a variable

def compound_interest(principal, rate, time):
    """
    Function used to calculate compound interest

    Args:
        - principal: amount of investment
        - rate: interest rate in percent
        - time: number of time periods interest is paid

    Returns:
        - Final amount under compound interest
    """
    return principal * (pow((1 + rate / 100), time))
